Multiplayer Game Platform Заключение

В результате выполнения проекта было спроектировано и реализовано приложение «Web-чат». В состав приложения входит необходимый минимум методов и функций, обеспечивающий полноценное функционирование объектов указанного класса. В процессе разработки приложения все поставленные задачи были выполнены, а именно:

  • проведен анализ предметной области, осуществили обзор существующих универсальных приложений и программных продуктов, в рассматриваемой предметной области, ни одно из существующего ПО не удовлетворяло всем требованиям;
  • разработан алгоритм работы приложения, на основе алгоритма создано Web-приложение
  • проработана структура приложения, разбито приложение на модули, каждый из которых выполнял одну из заявленных функций, каждый модуль проверен на правильную работоспособность;
  • выбран инструментарий, определена среда, в которой создано приложение и язык, на котором написан код приложения, среда разработки Visual Studio 2017, язык программирования С#;
  • написан основной код приложения и коды всех сопутствующих функций, код проверен;
  • разработан удобный для пользователя интерфейс приложения, с реализацией многооконного приложения;
  • разработано руководство пользователя, руководство программиста, обозначены системные требования;
  • протестировано полученное приложение, приложение работало корректно.

В дальнейшем возможно улучшение и расширение программы: может быть добавлено администрирование чата. Добавление функции обмена картинками, голосовыми сообщениями. Гибкая настройка профиля пользователя. Добавление других комнат по интересам. Вынесение проекта на интернет ресурс.

Данное приложение может быть доступно для пользователей на любом устройстве, умеющего доступ к интернету.

Цель данной работы полностью достигнута.
