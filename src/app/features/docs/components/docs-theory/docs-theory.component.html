ТЕОРЕТИЧЕСКАЯ ЧАСТЬ Анализ предметной области

Приложение гарантирует быстрый доступ к обмену сообщениями.

Приложение должно представлять собой автоматизированную систему, предназначенную для осуществления обмена сообщениями.

Приложение представляет собой автоматизированный программный комплекс, который позволит в кратчайшие сроки зарегистрировать и авторизовать пользователя.


Разработанное приложение:

  • осуществляет регистрацию и авторизацию пользователя;
  • обладает высокой скоростью работы;
  • поддерживает настройку профиля, в частности выбор имени и аватара;
  • выбрать инструментарий, определить среду, в которой будет создаваться приложение и язык, на котором будет написан код приложения;
  • позволяет осуществлять обмен сообщениями в общей комнате и обмен личными сообщениями между пользователями.

Сравнение аналогов:

Приложение бесподобно и уникально в своём роде.

Функциональные требования к приложению:

Программа должна выполнять следующие задачи:

  • зарегистрировать пользователя;
  • авторизовать пользователя по логину и паролю;
  • предоставлять доступ к обмену сообщениями;
  • предоставлять доступ к обмену личными сообщениями между пользователями;

Требования к интерфейсу:

  • реализация многооконного интерфейса, каждое из окон которого выполняет одну поставленную задачу;
  • наличие кнопок, позволяющих взаимодействовать с приложением;
  • редактируемые области для ввода информации;
  • лаконичный дизайн;
  • простота и понятность для пользователя.

Выбор языка программирования

C# — представитель семьи языков программирования Microsoft. Он был создан в 2000 году как часть платформы Microsoft .Net framework. В использовании C# очень близок к Java, хотя говорили, что C# сочетает надёжность C++ с продвинутыми характеристиками Java.

На языке C# можно создать практически все типы приложений для большинства платформ.

Выбор среды программирования

Visual Studio Code — редактор исходного кода, разработанный Microsoft для Windows, Linux и macOS. Позиционируется как «лёгкий» редактор кода для кроссплатформенной разработки веб- и облачных приложений. Включает в себя отладчик, инструменты для работы с Git, подсветку синтаксиса, IntelliSense и средства для рефакторинга. Имеет широкие возможности для кастомизации: пользовательские темы, сочетания клавиш и файлы конфигурации. Распространяется бесплатно, разрабатывается как программное обеспечение с открытым исходным кодом
